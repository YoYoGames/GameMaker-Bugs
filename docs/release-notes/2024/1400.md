# 2024.1400 Betas

[Required SDKs Wiki Page for 2024.14](https://github.com/YoYoGames/GameMaker-Bugs/wiki/2024.14)

<br>

2024.1400 is a range of separate Beta release collections (.0/.1/etc.) as we expand 2024.14 monthly to eventually be LTS26.0, so rather than have every release on one _enormous_ page you will find the correct release notes for the point release that you are using via the links below.

We **strongly** recommend using the latest release at all times, so treat the notes for older versions as legacy info.

<br>

If you are using an older version and you encounter a bug, always update to the latest version and check the bug still happens **before** reporting anything to us.
